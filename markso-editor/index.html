Features

Three editors.
One free plugin.

Write Markdown, edit raw XHTML, and export entire page trees — without leaving Confluence.

Live Markdown Editor

Write GitHub-Flavored Markdown in a split-pane editor with real-time preview. Supports tables, task lists, code blocks, Mermaid diagrams, collapsible sections, and more.

Interactive Task Lists

Checkboxes on published pages are clickable — check and uncheck tasks directly on the page. State saves automatically back to the macro, no re-editing required.

XHTML Source Editor

Edit the raw Confluence storage format with full XML syntax highlighting, live validation, find & replace, tag folding, minimap navigation, and a preview before save. Invalid XML can never be written to a page.

Markdown Export

Export any page or entire page hierarchy to clean Markdown files. Downloads as a structured zip mirroring the Confluence page tree. All conversion runs client-side — page content never leaves Atlassian.

Rich Formatting Toolbar

40+ formatting actions including text colour, font size, emoji picker, table size grid, alignment, highlight, superscript, subscript, Mermaid diagrams, and collapsible sections.

Dark Mode

Full VS Code-style dark theme for both the Markdown editor and Source Editor. One-click toggle, preference saved per user across sessions.

Runs on Atlassian

Zero external data egress. All compute on Atlassian Forge serverless infrastructure. All storage in Confluence or Forge KV Store. Your page content never touches any server outside Atlassian.
