QR Hub for Jira
Documentation

Getting started

QR Hub for Jira lets you generate scannable QR codes for any Jira issue, board, backlog, or URL — right inside Jira. This guide covers every feature, from installation to advanced print layouts.

📌 QR Hub for Jira is currently in development and not yet published on the Atlassian Marketplace. Documentation reflects the final feature set ahead of launch.

Installation

  1. Go to the Atlassian Marketplace and search for QR Hub for Jira.
  2. Click Get it now and select your Jira Cloud site.
  3. Review and accept the requested permissions.
  4. QR Hub appears automatically in issue sidebars, board menus, and under Apps in the left nav.

Installation takes under two minutes and requires no server configuration. The app runs entirely on Atlassian Forge.

Issue QR Panel

The Issue QR Panel is the most direct way to get a QR code for any Jira issue.

How to use

  1. Open any Jira issue.
  2. Scroll to QR Hub for Jira in the right sidebar panel.
  3. The QR code for that issue is generated automatically.
  4. Use the action buttons to Download, Print, or Copy the QR code.

The panel also displays the issue’s key, summary, type, status, project, and assignee alongside the QR code for quick reference.

Board & Backlog QR

Generate QR codes for your Scrum or Kanban boards and backlogs without leaving the board view.

How to use

  1. Open any board or backlog in Jira Software.
  2. Click the … (more actions) menu in the top right of the board.
  3. Select QR Code.
  4. A QR code for the current board or backlog URL is generated in a modal. Download or copy as needed.

Global QR Generator

Generate a QR code for any Jira URL from a central page, with automatic content-type detection.

How to use

  1. In the Jira left navigation, go to Apps → QR Hub.
  2. Paste any Jira URL into the input field.
  3. Click Generate.

QR Hub automatically detects the content type of the URL — issue, board, project, or generic link — and labels it accordingly so you always know what you’re sharing.

Dashboard Gadget

Persist a QR code on any Jira dashboard so your team can access it at a glance.

How to use

  1. Go to a Jira dashboard and click Edit.
  2. Click Add gadget and search for QR Hub.
  3. Add the gadget to your dashboard.
  4. Paste a Jira URL into the gadget’s input field and click Generate.
  5. The QR code is saved automatically and persists across page refreshes using Forge Storage.

🔒 The gadget URL is stored using storage:app inside Atlassian’s Forge infrastructure. No data leaves Atlassian.

Download formats

Every QR code in QR Hub can be exported in three formats and three sizes:

  • Formats: PNG, JPG, SVG
  • Sizes: 128px, 256px, 512px

SVG is recommended for print use as it scales to any size without loss of quality. PNG and JPG are better for embedding in documents, emails, or digital displays.

Print as PDF

QR Hub offers three distinct PDF print layouts:

  • QR Only — A clean, minimal layout with just the QR code. Ideal for quick labelling.
  • Full Issue Details — The QR code alongside a structured table of issue fields: key, summary, type, status, priority, reporter, assignee, and dates. Useful for printed handovers and asset documentation.
  • Jira Tag — A premium vertical layout styled like an airline boarding pass. Includes the QR code, issue key, summary, project, and status. Designed for physical asset tagging in warehouses, labs, or field environments.

To print, click the Print button from any QR panel, select your preferred layout, and use your browser’s print dialog or save as PDF.

QR styles

Customise the visual appearance of your QR codes with three dot styles:

  • Rounded — Soft, circular dots. Modern and approachable.
  • Classy — Elegant rounded squares. Professional and clean.
  • Square — Standard square dots. Maximum compatibility with all QR scanners.

The style selector is available in every QR generation context across the app. Your selection is applied immediately with live preview.

FAQ

Does QR Hub store my data?

Only the last URL entered in the dashboard gadget is stored, using storage:app inside Atlassian’s Forge infrastructure. No other data is persisted. No data of any kind leaves Atlassian.

Does scanning a QR code bypass Jira permissions?

No. The QR code contains only the Jira URL. All Jira access controls and permission schemes apply normally when the link is opened. If a user does not have access to the issue or board, Jira will handle that restriction as it normally would.

Which browsers are supported?

All modern browsers supported by Jira Cloud (Chrome, Firefox, Safari, Edge). No additional browser setup is required.

Does QR Hub work on Jira Server or Data Center?

No. QR Hub for Jira supports Jira Cloud only. It is built on Atlassian Forge, which is a Cloud-only platform.

Can I use the QR codes offline?

Once downloaded as PNG, JPG, or SVG, QR codes can be used offline in any context. The QR code itself is a static image — scanning it simply opens the encoded Jira URL in a browser, which does require internet access to load Jira.

Is QR Hub free?

Yes. QR Hub for Jira is completely free with no paid tiers, no feature gates, and no freemium limitations. All features are available to every user from day one.
